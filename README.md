## 🚀 What is YatraAI?
YatraAI is an AI-powered urban mobility dashboard that helps commuters navigate Indian cities smarter, faster, and safer — across Metro, Bus, Auto, Rapido, and Cab in one unified platform.

## ✨ Features
- **Multimodal Journey Planner** — Compare routes across 5+ transport modes
- **Real-time Transit Feed** — Live status of Metro, MTC, BMTC, KSRTC & more
- **AI Delay Prediction** — XGBoost model forecasting delays 6 hours ahead
- **Safety Layer** — Route safety scores + SOS alert system for women
- **Multi-city Support** — Chennai, Bangalore, Mumbai, Delhi
- **Mode Share Analytics** — Live breakdown of how the city is moving

## 🛠️ Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- XGBoost (delay prediction model)
- GTFS + Metro API data feeds
- Raspberry Pi + ESP32 (hardware roadmap)
